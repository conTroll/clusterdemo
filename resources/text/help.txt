	<h3>How to import and cluster a data set</h3>
	<ol>
		<li>Select <b>File &gt; Import</b></li>
		<li>Select a CSV file which contains your data.<br>The data should be encoded using UTF-8.</li>
		<li>Supply the separator character (sequence) used in your data file.</li>
		<li>Set if the attribute names are present in your data file.<br>
		This should take place in the first row of the data file.<br>
		If the attribute names aren't present the application will generate attribute names for you (A1, A2,..).</li>
		<li>Select <b>OK</b> to import the data file.</li>
		<li>Select a clustering algorithm and set the parameters on the right side of the application.</li>
		<li>Click on <b>Run Clustering</b> and wait for the process to finish.</li>
	</ol>
	
	<h3>How to use the visualization panel</h3>
	<ul>
		<li>Right click on the panel to add data.</li>
		<li>Right click on a node to edit or delete the data associated with it.</li>
		<li>In <b>MOVING</b> mode, hold down the <b>LEFT MOUSE BUTTON</b> to move the canvas.</li>
		<li>In <b>PICKING</b> mode, hold down the <b>LEFT MOUSE BUTTON</b> to select or move nodes.</li>
		<li>In <b>MOVING</b> mode, hold down the <b>SHIFT</b> key to rotate.</li>
		<li>In <b>MOVING</b> mode, hold down the <b>CTRL</b> key to skew the nodes.</li>
		<li>In <b>PICKING</b> mode, hold down the <b>SHIFT</b> key to add more nodes to your selection.</li>
	</ul>
	
	<p>Please see the documentation for other tips regarding the use of the application. :)</p>
